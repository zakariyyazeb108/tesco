HTML Introduction

HTML is the standard markup language for creating Web pages.

What is HTML?

  • HTML stands for Hyper Text Markup Language
  • HTML is the standard markup language for creating Web pages
  • HTML describes the structure of a Web page
  • HTML consists of a series of elements
  • HTML elements tell the browser how to display the content

A Simple HTML Document

<!DOCTYPE html>
<html lang="en">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>Page Title</title>
</head>
<body>
  <h1>My First Heading</h1>
  <p>My first paragraph.</p>
</body>
</html>
Note: The <!DOCTYPE html> declaration defines that this document is an HTML5 document. It must only appear once, at the top of the page.

HTML Elements

An HTML element is defined by a start tag, some content, and an end tag.

Element Syntax

<tagname>Content goes here...</tagname>

The HTML element is everything from the start tag to the end tag. Some HTML elements have no content (like <br>). These are called empty elements.

Nested Elements

HTML elements can be nested — elements can contain other elements. All HTML documents consist of nested elements.

<html>
<body>
  <h1>My First Heading</h1>
  <p>My first paragraph.</p>
</body>
</html>
Tip: Never skip the end tag. Some elements will display correctly even without the end tag, but you should never rely on this. Unexpected results may occur.

HTML Attributes

All HTML elements can have attributes. Attributes provide additional information about elements.

  • Attributes are always specified in the start tag
  • Attributes usually come in name/value pairs: name="value"

Common Attributes

AttributeDescription
hrefSpecifies the URL for a link
srcSpecifies the path to an image
altSpecifies alternate text for an image
classSpecifies a class name for styling
idSpecifies a unique id for an element
styleSpecifies inline CSS for an element
<a href="https://example.com" target="_blank">Visit Example</a>
<img src="photo.jpg" alt="A photo" width="500">

HTML Headings

HTML headings are defined with the <h1> to <h6> tags.

<h1> defines the most important heading. <h6> defines the least important heading.

<h1>Heading 1</h1>
<h2>Heading 2</h2>
<h3>Heading 3</h3>
<h4>Heading 4</h4>
<h5>Heading 5</h5>
<h6>Heading 6</h6>
Note: Browsers automatically add some white space (a margin) before and after a heading. Use only one <h1> per page for SEO best practices.

HTML Paragraphs

The HTML <p> element defines a paragraph.

A paragraph always starts on a new line, and browsers automatically add some white space before and after a paragraph.

<p>This is a paragraph.</p>
<p>This is another paragraph.</p>

Line Breaks

Use <br> for a line break without starting a new paragraph:

<p>This is<br>a paragraph<br>with line breaks.</p>

Horizontal Rules

The <hr> tag defines a thematic break and is displayed as a horizontal rule:

<h1>Heading 1</h1>
<p>Some text.</p>
<hr>
<h2>Heading 2</h2>

HTML Links

Links are found in nearly all web pages. Links allow users to click their way from page to page.

Syntax

<a href="url">link text</a>

The href attribute specifies the URL of the page the link goes to. The link text is the visible part.

Target Attribute

The target attribute specifies where to open the linked document:

  • _self — Opens in the same window/tab (default)
  • _blank — Opens in a new window or tab
  • _parent — Opens in the parent frame
  • _top — Opens in the full body of the window

HTML Images

The HTML <img> tag is used to embed an image in a web page.

<img src="pic.jpg" alt="Description" width="500" height="333">

Required Attributes

  • src — Specifies the path to the image
  • alt — Specifies alternate text for the image (important for accessibility)
Tip: Always specify the width and height of an image. If width and height are not specified, the web page might flicker while the image loads.

CSS Introduction

CSS is the language we use to style an HTML document. CSS describes how HTML elements should be displayed.

Why Use CSS?

CSS is used to define styles for your web pages, including the design, layout, and variations in display for different devices and screen sizes.

CSS Syntax

selector {
  property: value;
}

A CSS rule consists of a selector and a declaration block. The selector points to the HTML element. The declaration block contains one or more declarations separated by semicolons.

Try It Yourself

Example

CSS Selectors

CSS selectors are used to "find" (or select) the HTML elements you want to style.

Simple Selectors

SelectorExampleDescription
elementpSelects all <p> elements
#id#firstnameSelects element with id="firstname"
.class.introSelects all elements with class="intro"
**Selects all elements
element, elementdiv, pSelects all <div> and <p>
Specificity: Inline styles > IDs > Classes/Attributes > Elements. When two rules have equal specificity, the last one wins.

CSS Box Model

All HTML elements can be considered as boxes. The CSS box model is a box that wraps around every HTML element.

It consists of: margins, borders, padding, and the actual content.

div {
  width: 300px;
  padding: 10px;
  border: 5px solid gray;
  margin: 20px;
}

Total Element Width

Total width = width + left padding + right padding + left border + right border + left margin + right margin

Tip: Use box-sizing: border-box to include padding and border in the element's total width and height. Most developers set this globally with * { box-sizing: border-box; }

CSS Flexbox

The Flexible Box Layout Module makes it easier to design flexible responsive layout structures.

Flex Container

.container {
  display: flex;
  flex-direction: row;
  justify-content: space-between;
  align-items: center;
  gap: 16px;
}

Flex Properties

PropertyDescription
flex-directionDirection of flex items (row, column)
justify-contentHorizontal alignment
align-itemsVertical alignment
flex-wrapWhether items wrap to next line
gapSpace between items
flexShorthand for grow, shrink, basis

CSS Grid Layout

CSS Grid Layout offers a grid-based layout system with rows and columns, making it easier to design web pages.

.grid-container {
  display: grid;
  grid-template-columns: repeat(auto-fill, minmax(200px, 1fr));
  gap: 16px;
}

.grid-item {
  background: #f1f1f1;
  padding: 20px;
  text-align: center;
}
Tip: Use Flexbox for one-dimensional layouts (row OR column) and Grid for two-dimensional layouts (rows AND columns).

JavaScript Introduction

JavaScript is the world's most popular programming language. JavaScript is the programming language of the Web.

Why Study JavaScript?

JavaScript is one of the 3 languages all web developers must learn:

  1. HTML to define the content of web pages
  2. CSS to specify the layout of web pages
  3. JavaScript to program the behaviour of web pages

Try It Yourself

Example

JavaScript Variables

Variables are containers for storing data values.

Declaring Variables

let x = 5;
const PI = 3.14159;
var name = "John";  // older syntax

let vs const vs var

KeywordScopeReassignRedeclare
varFunctionYesYes
letBlockYesNo
constBlockNoNo
Best practice: Always declare variables with const. Only use let if you know the value will change. Never use var in modern code.

JavaScript Functions

A JavaScript function is a block of code designed to perform a particular task.

function greet(name) {
  return "Hello " + name + "!";
}
console.log(greet("World"));

Arrow Functions (ES6)

const greet = (name) => "Hello " + name + "!";
const square = x => x * x;

JavaScript DOM

The DOM (Document Object Model) is a programming interface for HTML documents.

Finding Elements

document.getElementById("myId");
document.querySelector(".myClass");
document.querySelectorAll("p");

Changing Content

document.getElementById("demo").innerHTML = "New text!";
document.getElementById("demo").style.color = "red";

JavaScript Events

HTML events are "things" that happen to HTML elements. JavaScript can "react" to these events.

<button onclick="alert('Hello!')">Click Me</button>

addEventListener

const btn = document.querySelector("button");
btn.addEventListener("click", function() {
  alert("Button was clicked!");
});

Common Events

EventDescription
clickUser clicks an element

JavaScript Arrays

An array is a special variable, which can hold more than one value.

const cars = ["Saab", "Volvo", "BMW"];
console.log(cars[0]);  // "Saab"
console.log(cars.length);  // 3

Array Methods

MethodDescription
push()Adds element to end
pop()Removes last element
map()Creates new array with results
filter()Creates new array with matching items
find()Returns first match

reduce()Reduces array to single value

JavaScript JSON

JSON (JavaScript Object Notation) is a format for storing and transporting data.

const text = '{"name":"John", "age":30}';
const obj = JSON.parse(text);
console.log(obj.name);  // "John"

JSON.stringify()

const obj = { name: "John", age: 30 };
const text = JSON.stringify(obj);
// '{"name":"John","age":30}'

JavaScript Async

Asynchronous JavaScript allows you to execute code without blocking the main thread.

Promises

fetch("https://api.example.com/data")
  .then(response => response.json())
  .then(data => console.log(data))
  .catch(err => console.error(err));

Async / Await

async function loadData() {
  try {
    const response = await fetch("/api/data");
    if (!response.ok) throw new Error(response.status);
    const data = await response.json();
    console.log(data);
  } catch (err) {
    console.error("Failed:", err);
  }
}
Tip: async/await is syntactic sugar over Promises. It makes asynchronous code look and behave like synchronous code.

SQL Introduction

SQL is a standard language for storing, manipulating, and retrieving data in databases.

What is SQL?

  • SQL stands for Structured Query Language
  • SQL lets you access and manipulate databases
  • SQL became a standard of ANSI in 1986, and ISO in 1987

What Can SQL do?

  • Execute queries against a database
  • Retrieve data from a database
  • Insert, update, and delete records
  • Create new databases and tables
  • Set permissions on tables and procedures

SQL SELECT Statement

The SELECT statement is used to select data from a database.

SELECT column1, column2
FROM table_name;
SELECT * FROM Customers;

SELECT DISTINCT

SELECT DISTINCT Country FROM Customers;

COUNT, AVG, SUM

SELECT COUNT(*) FROM Products;
SELECT AVG(Price) FROM Products;
SELECT SUM(Quantity) FROM OrderDetails;

SQL WHERE Clause

The WHERE clause is used to filter records.

SELECT * FROM Customers
WHERE Country = 'Mexico';

Operators

OperatorDescription
=Equal
<>Not equal
>Greater than
<Less than
BETWEENBetween a range
LIKEPattern matching
INMultiple possible values
SELECT * FROM Products
WHERE Price BETWEEN 10 AND 20
AND CategoryID IN (1, 2, 3);

SQL JOIN

A JOIN clause is used to combine rows from two or more tables, based on a related column.

INNER JOIN

SELECT Orders.OrderID, Customers.CustomerName
FROM Orders
INNER JOIN Customers ON Orders.CustomerID = Customers.CustomerID;

Types of Joins

JoinDescription
INNER JOINReturns matching rows in both tables
LEFT JOINReturns all rows from left table + matches from right
RIGHT JOINReturns all rows from right table + matches from left
FULL JOINReturns all rows when there is a match in either table

SQL INSERT INTO

The INSERT INTO statement is used to insert new records in a table.

INSERT INTO Customers (CustomerName, City, Country)
VALUES ('Cardinal', 'Stavanger', 'Norway');
Note: Always use parameterised queries (prepared statements) in real applications to prevent SQL injection attacks.

SQL UPDATE / DELETE

The UPDATE statement modifies existing records. The DELETE statement deletes records.

UPDATE

UPDATE Customers
SET ContactName = 'Alfred', City = 'Frankfurt'
WHERE CustomerID = 1;

DELETE

DELETE FROM Customers
WHERE CustomerName = 'Cardinal';
Warning: If you omit the WHERE clause in UPDATE or DELETE, all records will be affected!

Python Introduction

Python is a popular programming language, created by Guido van Rossum, and released in 1991.

What can Python do?

  • Web development (server-side)
  • Software development
  • Mathematics and data science
  • System scripting and automation

Python Syntax Example

print("Hello, World!")
Good to know: Python uses new lines to complete a command, and indentation to indicate a block of code. Python does not use curly brackets for blocks.

Python Syntax

Python syntax can be executed directly in the command line or in a .py file.

Indentation

Python uses indentation to indicate a block of code. The number of spaces is up to the developer, but at least one is required.

if 5 > 2:
  print("Five is greater than two!")

Comments

# This is a comment
"""
This is a
multi-line comment
"""

Python Variables

Variables are containers for storing data values. Python has no command for declaring a variable.

x = 5
y = "Hello"
print(x)
print(y)

Data Types

TypeExample
str"Hello"
int20
float20.5
boolTrue
list["a", "b", "c"]
dict{"key": "value"}

Python If...Else

Python supports the usual logical conditions from mathematics.

a = 200
b = 33

if b > a:
  print("b is greater than a")
elif a == b:
  print("a and b are equal")
else:
  print("a is greater than b")

Short Hand If

print("Yes") if a > b else print("No")

Python Loops

Python has two primitive loop commands: for and while.

For Loop

fruits = ["apple", "banana", "cherry"]
for x in fruits:
  print(x)

While Loop

i = 1
while i < 6:
  print(i)
  i += 1

Range

for x in range(6):
  print(x)
# 0, 1, 2, 3, 4, 5

Python Functions

A function is a block of code which only runs when it is called.

def my_function(name):
  print("Hello, " + name)

my_function("World")

Return Values

def multiply(x):
  return 5 * x

print(multiply(3))  # 15

Lambda

square = lambda x: x * x
print(square(5))  # 25
